flex 레이아웃
1. 부모 요소에 적용하는 속성
    - display : flex;
    
    - flex-wrap : 줄바꿈 지정
        1. nowrap : default / 줄바꿈 X
        2. wrap : 줄바꿈(오른쪽부터 아래줄로)
        3. wrap-reverse(왼쪽부터 아래줄로)
    
    - justify-content : 주 축의 정렬 방법
        1. flex-start : default / 왼쪽(위쪽) 정렬
        2. flex-end : 오른쪽(아래쪽) 정렬
        3. center : 가운데 정렬
        4. space-between : 시작과 끝 요소를 끝으로 밀고 남은 공백 나눠갖기
        5. space-around : 요소들 주위를 공백으로 채우기(공백이 겹치면 더 넓어짐)
        6. space-evenly : 공백을 똑같이 나눠갖기
    
    - align-items : 주 축의 교차(보조)축의 정렬 방식
        - 한 줄로 처리할 때 사용
        - x축: 주축 y축: 교차축

        1. flex-start : 각 줄의 시작점으로 정렬
        2. flex-end : 각 줄의 끝점으로 정렬
        3. center : 가운데 정렬
        4. baseline : 문자 기준선에 정렬

        - stretch : default / 자식 요소들이 부모 요소의 높이를 채움(자식 높이를 설정하지 않았을 때)
    - align-content  : align-items와 동작 방식 동일
        - 두 줄일 때 사용
        - align-items랑 같이 쓰는 경우 align-content를 우선함


    - flex-direction : 자식(items)을 어느 축으로 정렬할 것인가?
        1. row : default / items를 수평(좌 -> 우)축 기준으로 표시
        2. row-reverse : row의 반대 축으로 표시
        3. column : items를 수직(위 -> 아래)축 기준으로 표시
        4. column-reverse : column의 반대 축으로 표시

2. 자식 요소에 적용하는 속성
    - order : items 순서 설정  
        - 0 (default) 
        - 음수도 설정 가능하며 숫자가 클수록 뒤로 감
    
    - flex-grow : items 증가 비율 
        부모 요소에 '여백'이 있는 경우 아이템 너비를 늘릴 수 있다 
        - 0 (default)
        - 1:1:3 (높을수록 높은 비율을 가짐)
    
    - flex-shrink : items 감소 비율
        - 숫자가 클수록 많은 너비 감소 (더 작아진다)

    - flex-basis : items (공간 배분 전) 기본 너비 설정
        - auto (default) : width를 사용하거나 컨텐츠들의 크기에 비례


    -flex(flex-grow, flex-shrink, flex-basis)
        - flex : 1 1 0(크기 균등 계산) ;
    
    - align-self
